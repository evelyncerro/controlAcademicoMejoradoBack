
# Control Académico – API Backend (.NET 10 + MySQL)

Este proyecto es una **API REST desarrollada en ASP.NET Core 10 (Preview)** para la gestión
de un sistema de control académico (usuarios, asignaturas y horarios), usando **MySQL** como base de datos.

Sirve como backend para el proyecto frontend `control-academico-frontend`.

---

## 1. Requisitos previos

Antes de ejecutar el proyecto, asegúrate de tener instalado:

- **.NET SDK 10 Preview**
- **MySQL**
- **Visual Studio 2022** o **VS Code**

Verificar versión de .NET:

```bash
dotnet --version
